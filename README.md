# CryDeer
A CLI express package helper

## Usage
### Add New Package
```
crydeer add [tracking number] --description [desctription]
```

### Delete Package

```
crydeer delete [the first few digit of the number]
```

### Show Package Details

```
crydeer detail [the first few digit of the number]
```

### List Package

```
crydeer list
```

### Update Information

```
crydeer update
```
